# Examples

The `example/` directory contains an Expo Router app with 10 screens demonstrating all Drax features. Each example below includes a description, key concepts, and the full source code.

## Running the Examples

```bash
cd example && yarn start
```

Use `--clear` flag after library source changes to bust the Metro cache.

## Try It Live

Visit the [live interactive demo](https://nuclearpasta.com/react-native-drax/example/) to try the examples in your browser (Expo web export).

## Example Screens

| Example | Features |
|---|---|
| [Color Drag & Drop](/examples/color-drag-drop) | Free-form DnD, drop acceptance, hover styles, snap alignment |
| [Reorderable List](/examples/reorderable-list) | DraxList, animation presets, auto-scroll |
| [Reorderable Grid](/examples/reorderable-grid) | Sortable grid, multi-column, numColumns |
| [Drag Handles](/examples/drag-handles) | DraxHandle, only grip starts drag |
| [Drag Bounds](/examples/drag-bounds) | dragBoundsRef, constrained drag area |
| [Collision Modes](/examples/collision-modes) | Center vs Intersect vs Contain |
| [Kanban Board](/examples/kanban-board) | Cross-container drag, useSortableBoard |
| [Knight Moves](/examples/knight-moves) | Chess knight puzzle, conditional acceptance |
| [Scrolling](/examples/scrolling) | DraxScrollView, auto-scroll, drag from scroll |
| [Stress Test](/examples/stress-test) | 100 items in a sortable list |
