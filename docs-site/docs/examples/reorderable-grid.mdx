# Reorderable Grid

Demonstrates a sortable grid using `DraxList` with `numColumns`.

## Key Concepts

- **Grid layout** — `numColumns` creates multi-column grid
- **2D reorder** — items shift both horizontally and vertically
- **Same API** — identical to sortable list, just add `numColumns`

## Features Shown

| Feature | How |
|---|---|
| `numColumns` | Multi-column grid layout |
| `DraxList` | Same convenience wrapper as list |
| Slot detection | Position-based grid slot calculation |
